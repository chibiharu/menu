# Accordion_menu
jqueryを使用したシンプルなアコーディングメニューです。
 
# Requirement
HTML5/CSS3
JavaScript
jQuery v3.2.1 
 
# Usage
１：各ファイルを指定の場所に置き、HTMLに以下コードを記載
～～～
<head>
  <link rel="stylesheet" href="css/style.css">
  <script src="jquery.min.js"></script>
</head>
～～～

２：<body></body>内にjqueryコードを記載
※詳細はindex.html参照
 
# Note
移植はコードとjqueryをそのまま移せばいいだけなので簡単にできます
